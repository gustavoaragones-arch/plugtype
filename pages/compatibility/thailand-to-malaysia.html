Can You Use a Thailand Plug in Malaysia?

← Compatibility tool

Home → Compatibility Guides → Thailand → Malaysia

Compatibility summary

You will need a travel adapter in Malaysia; plug types used there differ from Thailand.

Plug comparison

Thailand plug types: A B C O

Malaysia plug types: G

There is no common plug type between Thailand (A B C O) and Malaysia (G). You need a travel adapter.

Voltage comparison

Thailand: 220V, 50Hz

Malaysia: 230V, 50Hz

Voltage difference is within a range many devices tolerate; check your appliance label.
